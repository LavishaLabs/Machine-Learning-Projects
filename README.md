#  Diabetes Prediction ML Web App

##  Project Overview
This project is a Machine Learning web application that predicts whether a person is diabetic or not based on medical input features.  
It is built using Python and deployed using Streamlit/Flask (edit based on your project).

---

##  Features
- User-friendly interface
- Real-time prediction
- Trained ML model
- Simple input form for medical data

##  Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Streamlit / Flask
- Pickle (for model saving)
